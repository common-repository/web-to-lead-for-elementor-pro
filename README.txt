Web-to-Lead for Elementor Pro is a Wordpress plugin which allows you to create new leads in Salesforce using the Elementor Pro Form widget.

== Description ==

In order to use this plugin you need a [Salesforce](https://salesforce.com/) account. By signing up for an account you agree to Salesforce's [Terms of Service](https://www.salesforce.com/company/legal/sfdc-website-terms-of-service/).
